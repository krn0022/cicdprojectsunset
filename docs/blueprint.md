# **App Name**: ForgeFlow CI

## Core Features:

- User Login Interface: A responsive login form for existing users, supporting basic input validation and form submission.
- New User Registration: A signup form to create new user accounts, including fields for username, email, and password, with client-side validation.
- Password Reset/Change UI: Forms for initiating password reset requests and for users to change their password securely.
- Dashboard/Protected Route: A placeholder landing page accessible only after 'successful' login, demonstrating route protection.
- Responsive Layout: The application's UI elements will adapt and display optimally across various device sizes (desktop, tablet, mobile).

## Style Guidelines:

- The theme utilizes a dark color scheme, providing a modern and developer-friendly aesthetic that minimizes eye strain during extended use.
- Primary color: A deep, authoritative blue-violet (#24248F) is used for key interactive elements like buttons and primary headings, conveying professionalism and technology.
- Background color: A subtle dark blue-gray (#1A1A1F) provides a stable and non-distracting canvas, allowing content and interactive elements to stand out clearly within the dark theme.
- Accent color: A vibrant sky blue (#33B1E3) is reserved for highlighting important information, links, and user feedback, ensuring critical elements draw immediate attention against the dark background.
- Body and headline font: 'Space Grotesk' (sans-serif) for its modern, slightly technical yet highly legible appearance, suitable for both prominent headlines and functional text.
- Use clear, concise vector-based icons that complement the clean UI and improve navigation for login, signup, and password management features.
- Employ a modular and structured layout using Flexbox and CSS Grid to ensure clean separation of components and efficient use of space across responsive breakpoints.
- Subtle and functional transitions for form submissions, navigation changes, and state updates to provide visual feedback and improve user experience without being distracting.
